# react-translation-app
React application that uses a translate API to translate input to another language!
<h1>API</h1>
<li>https://rapidapi.com/googlecloud/api/google-translate1/</li>


<h1>Dependencies</h1>
<li>react</li>
<li>react-router-dom</li>
<li>axios</li>
<li>tailwindcss</li>
<li>react</li>
<li>postcss</li>
<li>autoprefixer</li>


<h1>Script</h1>
npm start - runs app
